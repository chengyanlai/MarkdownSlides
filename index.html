Single band model with single X-ray photon event

  • Fermi Hubbard model in 1D (w/o core hole)

    $\mathcal{H}_{\text{FHM}}=-J\sum_{\langle ij \rangle,\sigma}(c^\dagger_{i\sigma}c_{j\sigma}+h.c.)+U\sum_in_{i\uparrow}n_{i\downarrow}\quad\Rightarrow\quad\mathcal{H}_{\text{FHM}}\vert GS\rangle=E_0\vert GS\rangle$

  • X-ray photon excites one electron from valance band to conduction band, and create a core hole which has effective attractive potential.

    $\vert\Psi(t=0)\rangle=c^\dagger_{ch}\vert GS\rangle;\quad\quad\mathcal{H}(t\geq0)=\mathcal{H}_{\text{FHM}}$$-V_{\text{ch}}n_{ch}$.

XAS for equilibrium state

  • The double-peak structure emerges from the core hole effect.
  • Single particle $\omega\sim -V_{ch}$ core hole effect!
  • Doubly occupied $\omega\sim U-2V_{ch}$.
  • Empty $\omega\sim0$ (inset).
  • Half-filled: insulating state $\Rightarrow$ dominant doubly occupied signal.
  • Away from half-filling: metallic state $\Rightarrow$ strongest signal at single particle.

Impurity induced metal droplet

Half-filling, $\Omega t_0=3$, and $V_{ch}=12J$.

  • $t\rightarrow te^{iA(t)}$ where $A(t)$ is Gaussian envelope.

    $A(t)\!=\!A_0e^{-(t-t_d)^2/2\tau^2}\cos\Omega(t-t_d)$

  • Frequency shifts: energy from pump pulse.
  • Weight changes: the doubly occupied peak are diminished by the pump pulse.
  • New peaks: response to the frequency of the pump pulse.
